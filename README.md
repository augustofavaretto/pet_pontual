# Pet Pontual

App para agendamento de banho e vacinas de pets. Esta é a estrutura inicial de um aplicativo Flutter gerada via Codex.

## Como começar

1. Certifique-se de ter o Flutter instalado e em seu `PATH`.
2. No terminal, rode `flutter pub get` para baixar as dependências.
3. Execute `flutter run` para iniciar o app em um emulador ou dispositivo.

## Organização

- `lib/`: Código-fonte principal do app.
- `assets/`: Recursos compartilhados (imagens, fontes).
- `test/`: Testes de widget.

Sinta-se à vontade para customizar a estrutura conforme necessário para o seu projeto.
